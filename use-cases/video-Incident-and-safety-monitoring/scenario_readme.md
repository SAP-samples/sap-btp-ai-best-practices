# Video Incident & Safety Monitoring - Usage Scenario

---

## Step 1: Select Media File

1. Select **Media Type** (Video or Audio)
2. Choose file from **"Select from Server"** or click **"Upload File"**
3. Preview appears automatically

## Step 2: Configure Analysis

1. Enter your prompt in **"Instruction for AI"** text area
2. Click **"Analyze Media"** button

## Step 3: Review Results

- **AI Analysis:** Findings and recommendations
- **Incident Detected:** Yes/No with severity level
- Click **"Download Report"** to save results
